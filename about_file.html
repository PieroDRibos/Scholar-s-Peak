GR

Στόχος της συγκεκριμένης εργασίας είναι η ανάπτυξη διαδικτυακής εφαρμογής για τη δημιουργία προφίλ ερευνητή με αντικείμενο την πληροφορική. Πιο συγκεκριμένα, ο χρήστης θα μπορεί να πληκτρολογεί το Google Scholar ID ενός εγγεγραμμένου χρήστη του Google Scholar στο πεδίο αναζήτησης και στη συνέχεια, θα λαμβάνει μία αναφορά με στατιστικά στοιχεία και γραφήματα τα οποία αφορούν τις “Αναφορές” που έχουν γίνει σε δημοσιεύσεις, οι οποίες περιλαμβάνουν το όνομα του συγκεκριμένου χρήστη, αλλά και τις “Δημοσιεύσεις” τις οποίες έχει αναρτήσει ο ίδιος.

Ενδεικτικά, χρησιμοποιήθηκαν οι παρακάτω τεχνολογίες για την υλοποίηση της εφαρμογής:

  • HTML5, CSS, και Bootstrap για την δημιουργία του οπτικού μέρους - σκελετού της εφαρμογής
  • JavaScript – (JQuery) για τη λειτουργικότητα του οπτικού μέρους όταν αλληλεπιδρά ο χρήστης με αυτό
  • AJAX για την επικοινωνία του Front End με το Back End κομμάτι
  • PHP και MySQL για την επικοινωνία με τη βάση δεδομένων
  • JSON για την επικοινωνία του Back End με το Front End
  • Symfony’s Mailer για την αποστολή αυτοματοποιημένων e-mail
  • Python για την εύρεση των δεδομένων του χρήστη του Google Scholar

EN

The aim of this project is the development of a web application for the creation of a researcher profile in the field of computer science. More specifically, the user will be able to type the Google Scholar ID of a registered Google Scholar user in the search field and then receive a report with statistics and graphs concerning the "References" made to publications, which include the name of the specific user, but also the "Publications" that he/she has posted.

Indicatively, the following technologies were used for the implementation of the application:

  • HTML5, CSS, and Bootstrap to create the visual part - the skeleton of the application
  • JavaScript - (JQuery) for the functionality of the visual part when the user interacts with it
  • AJAX for communication between the Front End and the Back End
  • PHP and MySQL for database communication
  • JSON for communication between the Back End and the Front End
  • Symfony's Mailer for sending automated e-mails
  • Python for finding Google Scholar user data
